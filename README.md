Lead Generation Automation

 Project Overview
This project is a Python automation script that collects and organizes lead data from a public source. It extracts important fields such as Name, Email, Website, and Location.

 🚀 Features
- Collects 20–30 lead entries
- Extracts Name, Email, Website, and Location
- Removes duplicate records
- Handles missing values
- Stores data automatically in Excel file
- Includes basic automation scheduling feature

 🛠 Technologies Used
- Python
- Pandas
- Requests
- BeautifulSoup
- Openpyxl
- Schedule

 📂 Output
The cleaned data is saved in:
leads.xlsx

 ▶ How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the script:
   python lead_scraper.py

The Excel file will be generated automatically.
